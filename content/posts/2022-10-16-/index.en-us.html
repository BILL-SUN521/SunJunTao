4.1

(a)

As the random variable X has pmd:\[p_1=\frac{1}{3},p_2=\frac{2}{3}\]

This can be accomplished by generating a random number U, and then setting

\[X=\begin{cases}1, if \; U < \frac{1}{3} \\ 2, if \; \frac{1}{3} \le U < 1 \end{cases}\] We use R programming language for simulation.

n=100
U=runif(n)
X=rep(0,n)
X[U<1/3]=1
X[U>1/3]=2
value<-sum(X==1)/n

According to the program, the proportion of values that are equal to 1 is 0.38

4.3

As the random variable X has pmd:\[p\{X=1\}=0.3,p\{X=2\}=0.2,p\{X=3\}=0.15,p\{X=4\}=0.35\]

This can be accomplished by generating a random number U, and then setting

\[X=\begin{cases}1, if \; U < 0.3 \\ 2, if \; 0.3 \le U < 0.5 \\3, if \; 0.5 \le U < 0.65 \\4, if \; 0.65 \; \le U < 1 \end{cases}\]

U=runif(1)
if (U<0.3) {
  X=1
}else if(U >= 0.3 & U <0.5) {
  X=2
}else if(U >= 0.5 & U <0.65) {
  X=3
}else if(U >= 0.65 & U <1) {
  X=4
}

According to the code, the value of that random variable X is:4
